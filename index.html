Instalando o GIT no MacOSX

  1. No Terminal: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/fabriziocucci/git-bash-for-mac/master/install.sh)"
  2. Reabra o Terminal antes de testar pela primeira vez

Comandos Shell para navegar entre pastas e arquivos

No terminal:

  • cd caminho_do_diretorio Vai para a pasta digitada. Ex: cd Desktop/app-projects/my-awesome-project
  • lsLista todos os arquivos da pasta navegada pelo comando cd
  • cd .. Navega até o diretório pai (parent). Ex: Navega de Desktop/app-projects -> Desktop
  • mkdir nome_da_pasta Cria uma nova pasta no diretório navegado
