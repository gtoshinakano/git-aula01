Instalando o GIT no MacOSX

  1. No Terminal: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/fabriziocucci/git-bash-for-mac/master/install.sh)"
  2. Reabra o Terminal antes de testar pela primeira vez

Comandos Shell para navegar entre pastas e arquivos

No terminal:

  • cd caminho_do_diretorio Vai para a pasta digitada. Ex: cd Desktop/app-projects/my-awesome-project
  • lsLista todos os arquivos da pasta navegada pelo comando cd
  • cd .. Navega até o diretório pai (parent). Ex: Navega de Desktop/app-projects -> Desktop
  • mkdir nome_da_pasta Cria uma nova pasta no diretório navegado
  • q Sair de comando

Dicionário GIT

  • repositório: Pasta local ou remota com todos os códigos fonte
  • stage: Palco das alterações antes de receber commit. O que está no stage está pronto para ser adicionado ao commit
  • commit: Salva o atual estado dos arquivos em stage e gera um log com as alterações
  • clone: Copia um repositório para uma pasta destino
  • push: Sobe os últimos commits ainda não atualizados no repositório remoto
  • pull: Baixa os últimos commits ainda não atualizados no repositório local
  • checkout: Volta todo o código para um commit anterior ou para um branch existente
  • .gitignore: Arquivo de texto que lista todos os arquivos que devem ser ignorados pelo GIT. Se o arquivo estiver na lista, não é possível adicioná-lo ao stage
  • master: Linha principal do repositório

Comandos GIT

No terminal:

  • git init: Inicia um repositório novo e vazio
  • git remote add origin http://.......url.git: Configura o endereço do repositório origem
  • git status: Verifica o status dos arquivos. Caso algum arquivo esteja alterado do último commit, ele é destacado aqui
  • git diff: Mostra as diferenças em todos os arquivos alterados e que não estão em stage.
  • git add . ou nome do arquivo: Adiciona os arquivos na stage para receberem o commit.
  • git commit -m ": Mensagem entre aspas"Mensagem ou título do commit
  • git log: Mostra uma lista com todos os commits do projeto do mais recente ao mais antigo
